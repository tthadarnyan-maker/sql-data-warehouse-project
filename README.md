# Data Warehouse and Analytics Project

Welcome to  the **Data Warehouse and Analytics Project** respository!
This project demonstrates a comprehensive data warehousing and analytics solution, from building data warehouse to generating to actionable insights. 


## Specifications
**Data Sources**: Import data from two source syatem (ERP and CRM) provided as csv files.
**Data Quality**: Clean and resolve data quality issues prior to analysis.

## BI: Analytics and reporting (Data Analytics)
### Objective
**Customer Behavior**
**Product Performance**
**Sales Trends**

These insights empower stakeholders with key business metrics, enabling strategic decision making.
